# **App Name**: Flashcard Focus

## Core Features:

- Card Creation: A form where students can enter the question and answer for each flashcard. Includes rich text formatting options.
- Library Persistence: Saves the created flashcards locally within the browser for permanent access, so all student data will remain after refreshing the page, closing the window, and even shutting down the computer.
- Offline Mode: Ensures full functionality without an internet connection for uninterrupted studying.
- Study Mode: Presents flashcards in a clear and focused study interface.
- Shuffle Feature: Shuffles the order of flashcards in study mode for varied review sessions.
- AI-Powered Content Suggestions: An AI tool that provides definitions and examples based on the content the student has entered on their card to improve or enhance understanding.
- Category tagging: Enable students to tag cards by content categories to easily find them and improve organization of the library.

## Style Guidelines:

- Primary color: Vivid purple (#9D2EC5) to convey a sense of focus, sophistication and effectiveness.
- Background color: Light gray (#F0F2F5), for a clean, distraction-free study environment.
- Accent color: Electric green (#7CFC00) to highlight key interactive elements, creating emphasis for study functions.
- Headline font: 'Space Grotesk', sans-serif, for headlines, giving a techy and modern feel.
- Body font: 'Inter', sans-serif, for body text, ensuring readability and clarity.
- Use a consistent set of line icons to represent different actions and categories, making navigation intuitive.
- Employ a clean, minimalist layout with clear visual hierarchy, ensuring the most important information stands out.
